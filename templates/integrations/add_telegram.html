Telegram

If your team uses Telegram, you can set up healthchecks.io to send status updates directly to the healthchecks telegrambot.

Setup Guide

1 Open your Telegram application and search for healthchecksbot. From the search results select healthchecksbot
2 Select start at the bottom of the application interface and you should receive a message entailing the chat ID. To complete telegram integration, copy and paste the chat ID into the chat ID field below..
3

If the start is not availabe run the command /start to receive the message.

Save the integration, and it's done!
